Mac OS X 安装说明

Mac OS X 10.5内置安装了Python 2.5.1版本。目前，Pygame只支持Python 2，并不支持Python。然而，本书的程序在Python 2和Python 3都可以运行。

在Python网站，在Mac上使用Python，提供了一些额外的信息。http://docs.python.org/dev/using/mac.html。
